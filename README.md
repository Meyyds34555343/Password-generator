# Password Generator GUI

A **simple password generator** built with **Python and Tkinter**.  
This tool allows you to generate **secure random passwords** with letters, digits, and symbols, and copy them to the clipboard easily.

---

## Features

- Generate random passwords of **any length** specified by the user.
- Passwords include **uppercase, lowercase, digits, and special characters**.
- **Copy password** to clipboard with a button click.
- **Error handling** for invalid or negative length input.
- **Clean and modern GUI** with a dark theme and easy-to-use layout.
